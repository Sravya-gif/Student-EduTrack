# Student-EduTrack
A Django-based Student ERP system with admin marks entry, student login, percentage calculation, and leaderboard.

Student EduTrack is a simple web-based Student ERP application developed using Django.  
It allows administrators to add student academic marks and enables students to view their subject-wise marks, total score, percentage, and leaderboard ranking using their roll number.

##  Features

- Admin login to add and manage student records
- Student login using roll number
- Subject-wise marks display (Maths, Physics, Chemistry)
- Automatic total and percentage calculation
- Leaderboard showing top 3 students
- Clean and responsive user interface

##  Technologies Used

- Python 3
- Django
- HTML, CSS
- SQLite (for development)

##  Steps to Run
1. Enter path of the project
2. python -m venv venv
3. venv\Scripts\activate
4. pip install django
5. python manage.py makemigrations
6. python manage.py migrate
7. python manage.py createsuperuser
8. python manage.py runserver
